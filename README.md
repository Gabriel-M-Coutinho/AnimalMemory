# 🐾 Animal Memory

Bem-vindo ao repositório oficial do **Animal Memory**! 🎮

Este é um jogo de memória clássico, desenvolvido com a engine **Godot 4**, focado em diversão e aprendizado sobre diferentes animais.

## 🛠️ Detalhes Técnicos

- **Engine**: Godot v4.x (GL Compatibility)
- **Orientação**: Paisagem (Landscape)
- **Física**: Jolt Physics (integrado para máxima performance)

## 📥 Como Jogar

1. Baixe o arquivo correspondente ao seu sistema operacional abaixo (`.apk` para Android ou o executável para desktop).
2. Instale/Extraia e abra o jogo.
3. Tente encontrar todos os pares de animais no menor tempo possível!
